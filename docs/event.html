Event List

You can open the Event List window from the toolbar (Edit > Event List...) or by pressing CTRL+L while having a selected track or part that is of a supported type (only Midi or Drum Tracks I think).

This will open a window that contains a list with all Note, SysEx, Meta, and Control events in the selected part (or the whole track if no part is selected when opening the window).

The screenshot above shows the Event List for Track 2, its list contains 12 events, 10 of which are Note events, 2 of which are Ctrl events of the "Program Change" type.

The event list shows:

  • The time at which the event happens (Tick, Bar)
  • The Type of the event, which can be: Note, SysEx, Ctrl, or Meta
  • The Channel in which the event happens, in this case 2 because the Midi track is set to channel 2.
  • Three Value fields which vary according to the type of event, for a Note event these correspond to its Note (D4), NoteOn Velocity, and NoteOff Velocity.

Creating an event

Note
You can create a Note event by pressing the Note button. There you can set its time, length, pitch and velocity. Drawing events in the piano roll is probably easier though.

SysEx
You can create a SysEx event by pressing the SysEx button. SysEx events are less common than the others, you can learn more about SysEx online.

Ctrl
You can create a Ctrl event by pressing the Ctrl button. With Ctrl events you can tweak things like the Program (Instrument), Pan, Reverb, Chorus, or even Pitch Wheel events too. In the example above we used two Ctrl events to change the instrument of the current channel twice.

Meta
You can create a Meta event by pressing the (you guessed it) Meta button. Meta events can hold things like Lyrics, Markers, etc.

Moving, Deleting, or Editing Events

By using the Tick+ and Tick- buttons, you can increase or decrease the tick position or the selected event(s) by one.

And by using the Delete button you can delete selected events.

As for editing an existing event, you can double click any event to edit it.
